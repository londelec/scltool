=======================================
Installation:
=======================================
System requirements:
  Java Runtime Environment (JRE) version 7 or later is required.

Copy 'scltool.jar' package and 'lib' directory to a desired location.


=======================================
Run:
=======================================
On Windows:
  Run SCL tool by double-clicking on 'scltool.jar',
  no Administrator privileges are required.

  Alternitevly open Command Prompt and enter:
  java -jar scltool.jar


On Linux:
  Run SCL tool by double-clicking on 'scltool.jar' or
  from command line:
  java -jar scltool.jar


SCL tool version V3.0 and above
